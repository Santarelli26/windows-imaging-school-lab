## Overview
This project documents the process of imaging, deploying, and restoring Windows systems in a school or enterprise environment.

## Operating Systems
- Windows 10
- Windows 11

## Imaging Tools
- Clonezilla
- Windows Media Creation Tool
- Microsoft Deployment Toolkit (conceptual)

## Imaging Workflow

### Step 1: Pre-Image Checklist
- Backup user data
- Verify hardware compatibility
- Confirm BIOS boot mode (UEFI)

### Step 2: OS Installation
- Clean Windows install
- Driver installation
- Windows updates

### Step 3: Post-Image Configuration
- User account setup
- Network configuration
- Printer installation
- Security updates

---

## Common Imaging Issues & Fixes

| Issue | Cause | Resolution |
|-----|------|------------|
| Boot failure | Wrong boot mode | Switch to UEFI |
| Missing drivers | OEM drivers | Install manually |
| Network unavailable | Driver missing | Update NIC driver |
